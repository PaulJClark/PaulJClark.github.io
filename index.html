Paul's HTML Cheatsheet

HTML (HyperText Markup Language) is used to give content to a web page and instructs web browsers on how to structure that content.

Elements and Structure

<ul> Creates an unordered list with bulletpoints
<ol> Creates an ordered list with numbering
<li> Creates an individual list item in an unordered or ordered list
<p> Creates a paragraph element with a block of text
<h1-h6> Six different levels of headings are available in html
<br> Creates a line break in text
<em> Creates text in italics
<strong> Creates text in bold
<video> Embeds a media player for video
<div> Short for division, a container used to separate an HTML document
